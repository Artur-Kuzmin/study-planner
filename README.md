# Avkast

An AI-native wealth management and investment platform built to help users manage portfolios, track markets, and receive intelligent financial guidance.

## Target Users

Retail investors, digital-native professionals, and users looking for a more personalized and proactive investing experience.

## Features

- AI-assisted portfolio insights
- Multi-agent market research system
- Real-time portfolio monitoring
- Explainable recommendation logic
- Portfolio rebalancing support
- Watchlist and asset tracking
- Scenario simulation for financial planning
- Contextual memory for user preferences and investment constraints

## Tech Stack

- Next.js
- React
- TypeScript
- PostCSS

## User Stories

- As an investor, I want to see my portfolio allocation and performance in one place, so that I can understand my current financial position.
- As an investor, I want to get AI-powered recommendations based on market data and my profile, so that I can improve my investment decisions.
- As an investor, I want each recommendation to include a clear explanation, so that I know what factors influenced it.
- As an investor, I want to save my preferences such as risk level and excluded sectors, so that recommendations stay aligned with my strategy.
- As an investor, I want to keep a watchlist of assets and monitor live changes, so that I can react to important market movements.
